How a crest is chosen. Teams type their own name on join (Mill Lane, North Store, whatever they like). That name is hashed to a number which picks an outer shape (shield / hexagon / circle / rounded square / diamond) and an inner icon from a store-manager-relevant set (storefront / trolley / basket / price tag / scales / key), plus a small accent mark (dot, stripe, dashed ring, corner tick, underline, bar). Every icon says something about running a store: the building itself, customers moving through it, what's being sold, how it's priced, the judgement behind the call, the custodianship of opening and closing. Same name always makes the same crest, so a team can recognise themselves at round start and in the post-session report. Everything is monochrome with the single orange accent, so it sits naturally in the existing palette and reads fine for a colour-blind viewer because differentiation is by shape and icon, not hue.
